## 📖 About the Project

**Uzhavar Mitra** is a free, offline-capable web app built to empower Tamil Nadu farmers with smart agricultural tools — in their own language. The entire app runs as a **single HTML file**. No installation, no server, no technical knowledge needed.

Tamil Nadu farmers often lack easy access to real-time crop prices, weather, expert advice, and government scheme information in Tamil. Uzhavar Mitra brings all of this into one simple, mobile-friendly app that works even in low-connectivity rural areas.

## ✨ Features

| Feature | Description |
|---|---|
| 🤖 **AI Chatbot** | Ask farming questions in Tamil or English — powered by Claude AI |
| 🌦️ **Live Weather** | Real GPS-based weather for your exact farm location |
| 💹 **Market Prices** | Live mandi prices from Thanjavur, Coimbatore, Trichy, Erode & more |
| 🌱 **Crop Guide** | 40+ Tamil Nadu crops across 8 categories with full care tips |
| 📓 **Farm Diary** | Log daily farm activities — saved privately on your device |
| 🧪 **Soil Calculator** | Fertilizer dosage based on crop type, soil, and farm size |
| 🐛 **Pest Guide** | Identify & treat common pests and diseases |
| 📅 **Harvest Calendar** | Season-wise planting schedule for Tamil Nadu |
| 🏛️ **Govt Schemes** | Tamil Nadu & central government schemes for farmers |
| 🌐 **Bilingual** | Full Tamil (தமிழ்) + English toggle |

## 👨‍🌾 Who Is It For?

Small and marginal farmers of Tamil Nadu who use Android smartphones and need reliable agricultural information in Tamil — without depending on middlemen or agents.
